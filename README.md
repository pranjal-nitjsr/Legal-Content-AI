
 Legal Content AI - AI-Based Content Generation System

Legal Content AI is an AI-based content generation system that creates simple legal service page content, FAQs, and client information drafts.

 Tech Stack
- Python
- Flask
- JavaScript
- REST APIs
- LLM API
- SQLite

 Features
- Takes user input like service type, audience, location, and tone
- Generates structured AI-based content
- Uses prompt templates for better output
- Simple frontend using HTML, CSS, and JavaScript
- Stores generated content history in SQLite

 My Role
I built the backend APIs, connected the LLM API, created prompt templates, developed the simple frontend, and stored generated content in SQLite.

 Challenge Faced
The main challenge was getting clean and useful AI-generated content. I improved this by using structured prompts and clear input fields so that the output became more professional and easy to review.

 What I Learned
- How to use prompt templates
- How to connect frontend with backend APIs
- How to store generated content history
- How LLM-based tools can be used in practical software products
